Unread mail folder


To create an unread mail folder:

1. In evolution 3.22.6:
select Edit > Search Folders

2. At Search Folders
click Add

3. At Add Rule

Rule name:              00Unread
Find items which match: all the following conditions
Include threads:        None

Added rule:
  Status is not Read

Search Folder Sources
  x Automatically update on any source fold change

  All active local folders
  All active remote folders
  All localt and active remote folders <===
  Specific folders

click OK

This will create the 00Unread search folder under "Search Folders"
